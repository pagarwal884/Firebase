# **App Name**: Data Intern

## Core Features:

- User Registration: Allows students to create an account with their name, email, mobile number, password and confirmation password.
- User Login: Enables registered users to log in using their email and password.
- Internship Listings: Displays available internships in various domains such as Data Science, Analytics, Machine Learning, Web Development, and Software Development with details like descriptions and requirements.
- Application Submission: Enables students to apply for internships by completing their profile and choosing their preferred internship domain.
- Domain Suggestions: AI tool that uses the applicant's background, interests, and resume information to suggest an internship domain.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey professionalism and trust.
- Background color: Light gray (#F5F7FA) to maintain a clean, light aesthetic.
- Accent color: Teal (#26A69A) to highlight CTAs and important information, providing a modern touch.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look.
- Maintain a clean and structured layout with consistent spacing, padding, and margins to enhance readability and navigation.
- Utilize simple, consistent icons to represent internship domains and benefits, ensuring clarity and visual appeal.
- Implement subtle animations, such as smooth scrolling and hover effects, to enhance user experience without being distracting.
